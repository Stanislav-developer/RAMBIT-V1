## 📋 Про проєкт

**RAMBIT** — мобільна робототехнічна платформа власної розробки для участі у відкритих обласних змаганнях **«RoboRace 2026»**.

Проєкт розроблено з нуля: від CAD-моделювання шасі до прошивки та електричної схеми. Використання готових RTR-рішень або серійних RC-шасі не допускається — вся конструкція є оригінальною розробкою.

---

## ⚙️ Технічні характеристики

| Параметр | Значення |
|---|---|
| Маса (з АКБ) | ≤ 1,4 кг |
| Габарити | ≤ 300 × 300 × 300 мм |
| Тип приводу | Електричний (Brushless DC) |
| Живлення | LiPo 2S / 3S / 4S (макс. 16,8 В) |
| Керування | Радіоканал 2.4 GHz |
| ESC | Дискретний, з підтримкою Fail-safe |
| Виготовлення | 3D-друк + лазерне різання |

## 📄 Інженерна документація

Відповідно до регламенту, проєкт містить мінімум два підтверджуючих документи:

- **CAD-моделі** — 3D-моделі шасі та вузлів (`hardware/3d models/`)
- **Електрична схема** — повна схема з'єднань (`hardware/electronics/`)
- **BOM** — специфікація компонентів та матеріалів (`docs/BOM.md`)
- **Розрахунки** — передаточне число, центр ваги, енергоефективність
