# 🚗 Car for RoboRace 2026

> Проєкт автономного гоночного автомобіля для участі у змаганнях **RoboRace 2026**.  
> Репозиторій містить прошивку мікроконтролера, принципові схеми, керуючий код та 3D-моделі корпусу й деталей.

---

## 📁 Структура репозиторію

```
Car-for-Roborace-2026/
│
├── docs/                          # Документація проєкту
│   └── Materials for presentation/  # Матеріали для захисту проєкту
│       ├── BOM.md                 # Специфікація компонентів (Bill of Materials)

│
├── hardware/                      # Апаратна частина
```

## 📝 Опис проєкту

Цей проєкт розроблено командою для участі у змаганнях **RoboRace 2026** — змаганнях автономних роботизованих автомобілів.  

Автомобіль побудований на базі шасі масштабу 1:10 з власною електронікою керування. Основні завдання — автономне проходження траси, уникання перешкод та максимальна швидкість у межах правил змагань.

### ✨ Ключові особливості

- 🧠 Автономне керування на базі мікроконтролера
- 📡 Сенсорна система: ультразвукові / інфрачервоні датчики (або камера)
- ⚙️ Власна PCB для керування двигунами та сервоприводом
- 🖨️ 3D-друковані деталі корпусу та кріплень під електроніку
- 📦 Повна документація: схеми, BOM, нотатки

## 🛠️ Технологічний стек

| Компонент | Технологія |
|---|---|
| Мікроконтролер | Arduino / ESP32 / STM32 |
| Середовище розробки | PlatformIO / Arduino IDE |
| Схемотехніка | KiCad / EasyEDA |
| 3D-моделювання | Fusion 360 / FreeCAD |
| Керуючий код | C++ / Python |

## 🚀 Початок роботи

1. Клонуй репозиторій:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Car-for-Roborace-2026.git
   ```
2. Відкрий папку `firmware/` у PlatformIO або Arduino IDE.
3. Підключи мікроконтролер та завантаж прошивку.
4. Деталі збірки та схеми — у папці `hardware/` та `docs/`.
